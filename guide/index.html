# 介绍

在高德地图中展示 echarts 图表，当前版本 2.0.0。

1.x 版本文档：https://liuweigl.github.io/amap-echarts-doc-v1/

# 它是如何工作的？

  1. 使用高德地图自定义图层为 echarts 提供绘制容器。
  2. 使用高德地图 api 解析经纬度数据为 px 坐标。
  3. 接管高德地图插件的引入与图层的更新事件保证 echarts 同步绘制。

# Features

  1. 灵活：amap 对象由外部传入，方便与 vue-amap 或者其他框架的 amap 组件配合使用。
  2. 轻量：类库内部只是接管了高德自定义图层的加载、刷新时机、经纬度转换功能，echarts 可按需加载。
  3. 一致：除了 Promise 化，实例接口与 echarts 实例保持一致，减轻 api 负担。
